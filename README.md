# Amazon Clone (HTML & CSS)

This is a simple Amazon homepage clone created using only *HTML* and *CSS*.

## Features
- Amazon-like homepage layout  
- Fully responsive design  
- Clean and structured UI  
- No JavaScript used  

## Technologies Used
- HTML5  
- CSS3  

## Project Structure
- index.html  
- style.css  
- /images (all project images)

## How to Run
Just open the *index.html* file in any browser.
